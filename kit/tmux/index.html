    tmux基本技巧¶

    会话session¶

    创建一个tmux的会话¶

    1
2
    tmux
tmux new -s name

    退出tmux会话¶

    1
2
3
4
5
    #后台运行
tmux attach
ctrl+b d
#退出不后台运行
exit

    列出所有的tmux会话¶

    1
2
    tmux ls
tmux list-sessions

    装载一个会话¶

    1
2
    tmux attach
tmux attach -t name

    销毁一个会话¶

    1
    tmux kill-session -t name

    窗口windows¶

    创建一个新窗口¶

    1
    crtl+b c

    窗口间切换¶

    1
2
3
    ctrl+b n(next的意思)
ctrl+b p(previous)
ctrl+b 0/1/2...

    关闭窗口¶

    1
2
    exit
ctrl+b &
